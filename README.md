# QLog

QLog is an Amateur Radio logging application for Linux, Windows and Mac OS. It
is based on the Qt 5 framework und uses PostgreSQL as database backend.

## Features

- ADIF import/export
- DX cluster integration
- Grey-line map
- HamQTH integration
- Rig control via hamlib
